# BetterSocialEmbeds

A **Shelter** plugin that automatically intercepts outgoing chat messages and replaces standard media URLs with direct-embedding alternatives right before they hit the network.

## Features
* Replaces `twitter.com` and `x.com` links with `fixupx.com`
* Replaces `instagram.com` links with `vxinstagram.com`
* Replaces `tiktok.com` links (including mobile short links like `vt.` or `vm.`) with `tnktok.com`

## How to Install

1. Copy the GitHub Pages URL (see below).
2. Open your client settings panel.
3. Scroll down to the **Shelter** settings segment and click on **Settings**.
4. Click on **Add Plugin**, then paste the installation URL line directly into the proxy install field box.
5. Click **Fetch**, and flip the toggle switch to active!
